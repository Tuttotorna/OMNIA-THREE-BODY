# OMNIA-THREE-BODY Scope

## Purpose

OMNIA-THREE-BODY is an experimental structural dynamics layer.

It uses three-body trajectories as a controlled stress test for structural divergence, perturbation sensitivity, and representation-dependent stability.

## Boundary

```text
measurement != inference != decision
structural dynamics != physical law
trajectory divergence != exact future prediction
```

## What is in scope

This repository measures:

- structural divergence time
- residual coherence
- irreversibility
- perturbation sensitivity
- trajectory-level collapse
- cross-representation divergence
- stability of invariant-like representations

## What is out of scope

This repository does not claim:

- solution of the three-body problem
- new physics
- exact future-state prediction
- physical simulation authority
- universal forecasting
- semantic interpretation
- decision authority

## Correct interpretation

```text
TΔ = structural divergence time
Ω  = residual structural coherence
IRI = accumulated irreversibility
representation stability != semantic truth
```
